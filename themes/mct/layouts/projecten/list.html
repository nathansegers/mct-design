2MCT - Project 2

Team Project

Halverwege het tweede jaar valt project 2. Het voornaamste doel in dit project is om de rollen & werkmethodes in projectmanagement op een agile manier toe te passen. Dit is de eerste keer in de opleiding dat een groot project in team uitgewerkt wordt. Het is ook een gelegenheid om je te verdiepen in datgene wat je echt boeit, met een aantal technische uitdagingen...

Hier werken we aan IT-projecten die we krijgen van andere Howest-opleidingen.

Projecten van halverwege het tweede jaar.

2MCT - Project 3

Industry Project

Op het einde van het 2de jaar werk je 4 weken lang in een team van 3 tot 5 MCT-studenten op een bedrijfscase aangeleverd door een Belgische of internationale onderneming. Je integreert verschillende inhouden en vaardigheden van verschillende opleidingsonderdelen in dit project. Bedrijven waar we reeds mee samenwerken zijn onder andere TVH, Smappee, Alvey Group, Ordina, ...
